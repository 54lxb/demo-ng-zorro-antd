# demo-ng-zorro-antd

##  示例运行步骤：
> 1. 克隆到本地
> 2. 打开文件夹
> 3. 安装依赖包：`npm install`
> 4. 本地运行：  `npm start`
> 5. 打开浏览器输入地址：  `http://localhost:4200`
